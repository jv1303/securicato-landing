Requisitos do Sistema

  • Python 3.10+ / Docker e Docker Compose
  • MySQL 8.0 ou PostgreSQL 13+
  • 4GB RAM mínimo (recomendado 8GB)
  • Sistemas operacionais: Linux (Ubuntu/Debian), Windows Server, macOS

Como instalar (Docker)

git clone https://github.com/securicato/securicato-platform.git
cd securicato-platform
docker-compose up -d

Acesse http://localhost:8000 e finalize a configuração via CLI.
